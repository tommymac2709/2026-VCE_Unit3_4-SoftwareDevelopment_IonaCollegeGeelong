# d. Applied Task

## Scenario

You're building a module that stores student results and needs to:

- load existing results when the program starts
- allow the user to add a new result
- save the updated list back to the file

---

## Task

Choose the most appropriate data source (TXT, CSV or XML) and justify your choice using:

- structure
- reasons for use

Explain how your chosen file format would store multiple records.

Explain one disadvantage of the other two formats for this scenario.

---

!!! warning "40+ tip"
    Your justification must **compare formats**, not just describe one.
